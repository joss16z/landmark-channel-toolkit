# Landmark Channel Toolkit

## How to update your deployed Streamlit app

1. Open your GitHub repository.
2. Replace `app.py` with this new version.
3. Replace `requirements.txt` if needed.
4. Commit changes.
5. Streamlit will redeploy automatically.
